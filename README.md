# Swift App Development Bootcamp Project Work

Questa repository contiene il progetto sviluppato durante il corso Swift App Development Bootcamp, volto allo sviluppo di applicazioni iOS, offerto dall'Università degli Studi di Salerno.

## Overview sul Progetto

Il progetto ha come obiettivo dimostrare l’apprendimento della programmazione per sistemi iOS e la conoscenza dei software messi a disposizione da Apple.
L’applicazione sviluppata si chiama Doggofinder e ha lo scopo di facilitare il ricongiungimento tra cani smarriti e i loro padroni.
Per raggiungere questo obiettivo, Doggofinder utilizza diverse tecnologie basate sia sulla geolocalizzazione sia sull’intelligenza artificiale.

Per maggiori informazioni consultare la documentazione.
